# GraamSeva — Government Schemes Finder 🏡

A web application to help elderly village residents discover 
government welfare schemes they qualify for.

## Features
- Filter schemes by age, gender, occupation and income
- 10+ real government schemes listed
- Category filter tags (Agriculture, Health, Pension etc.)
- Mobile responsive design
- Both CLI and Web versions available

## How to Use
**Web version:** Open `schemes_finder.html` in any browser  
**CLI version:** Run `python schemes_finder.py` in terminal

## Schemes Covered
PM Kisan | Ayushman Bharat | NREGA | PM Awas Yojana | 
Old Age Pension | Fasal Bima | Ujjwala Yojana and more

## Tech Used
- Python (CLI version)
- HTML, CSS, JavaScript (Web version)
